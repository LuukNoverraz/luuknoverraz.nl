Master Alein

Master Alein is a game I made for the Unity-0 module in the Coderclass. It is a 2D game, with all the sprites being made by me. The premise of the game is that you can't jump, instead, you use the environment to your adventage. First I had made a singeplayer test build, but I later redesigned the character and changed it into a two player game of tag. You can find the source code at this repository.
